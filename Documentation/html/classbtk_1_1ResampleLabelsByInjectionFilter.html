Detailed Description

template<class TInputImage, class TOutputImage, class TInterpolatorPrecisionType = double>
class btk::ResampleLabelsByInjectionFilter< TInputImage, TOutputImage, TInterpolatorPrecisionType >

Resample an image via a coordinate transform.

ResampleLabelsByInjectionFilter resamples an existing image through some coordinate transform, interpolating via some image function. The class is templated over the types of the input and output images.

Output information (spacing, size and direction) for the output image should be set. This information has the normal defaults of unit spacing, zero origin and identity direction. Optionally, the output information can be obtained from a reference image. If the reference image is provided and UseReferenceImage is On, then the spacing, origin and direction of the reference image will be used.

Since this filter produces an image which is a different size than its input, it needs to override several of the methods defined in ProcessObject in order to properly manage the pipeline execution model. In particular, this filter overrides ProcessObject::GenerateInputRequestedRegion() and ProcessObject::GenerateOutputInformation(). : 105
