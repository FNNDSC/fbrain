Detailed Description

template<typename TOutput = double, unsigned int VImageDimension = 3, typename TInput = Point<double, VImageDimension>>
class btk::OrientedSpatialFunction< TOutput, VImageDimension, TInput >

N-dimensional oriented spatial function class.

This class implements a function oriented in a specific direction. This is used to have the PSF in the correct orientation in world coordinates.
